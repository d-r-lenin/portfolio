# portfolio

## To run frontend 
 * goto ./client folder
 * run `npm i` to install dependencies
 * run `npm start` to run dev server on port 3000

## env variables
 * create a .env file in the root directory
 * add the following variables
 * EMAIL_PASSWORD=********
 * EMAIL_USERNAME=********
 * GOOGLE_SERVICE_ACCOUNT_KEY=********< json contents of the key file>
